# Memoria_V2.X.X
主要功能：导出B站平台的.m4s缓存文件为.mp4视频文件。将在后续开发支持“手机数据线访问”、“多平台”、“预览”、“批量导出”等新功能
预计将在未来开发的新功能：
1.通信端：手机-电脑数据线访问导出模式看，支持直接“将整个缓存原文件复制粘贴进电脑”功能
  1.1未来或许会开发“WiFi直接访问”功能，但需要深度学习
2.解析函数：支持解析缓存文件中的“entry.json”功能（包含大小、标题等视频数据）
  2.1支持多平台缓存文件解析适配（其他平台的缓存音视频应该也是FFmpeg形式的吧？“混流”部分应该不需要大改吧？？？）
3.预览界面：支持在导出前直接查看“预览”视频的功能，并根据标题同步在平台搜索链接，允许“跳转”查看（是否换源或被下架）
4.批量操作：支持“批量导入”与“批量导出”的操作
5.多格式支持：导出视频除了.mp4格式之外，未来将适配.mkv等其他格式的视频文件
6.视频下载（边缘）：与Downkyi、闪豆一样的“视频下载功能”（别太复杂，毕竟是针对于“已下架的缓存”的软件）
